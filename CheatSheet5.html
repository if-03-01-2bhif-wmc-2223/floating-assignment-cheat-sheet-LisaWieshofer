Graphic Formats

JPEG (Joint Photographic Experts Group):

JPEG is a good choice for photographs and continuous-tone images because it supports lossy compression, which can significantly reduce file size without a noticeable loss in quality.

  • Kind of graphical content: photograph, continuous-tone image
  • Compression: lossy (some loss of quality)
  • 'Motion picture' support: no

GIF (Graphics Interchange Format):

GIF is a good choice for simple graphics and animations because it supports lossless compression for graphics and can support simple animations.

  • Kind of graphical content: graphic, simple animations (limited to 256 colors)
  • Compression: lossless (no loss of quality) for graphics, lossy for animations
  • 'Motion picture' support: no

PNG (Portable Network Graphics):

PNG is a good choice for graphics and photographs that require lossless compression or transparent backgrounds.

  • Kind of graphical content: graphic, photograph
  • Compression: lossless (no loss of quality)
  • 'Motion picture' support: no

ICO (Icon file format):

ICO is a good choice for small graphics and icons.

  • Kind of graphical content: small graphics, icons
  • Compression: varies (can be lossless or lossy)
  • 'Motion picture' support: no

SVG (Scalable Vector Graphics):

SVG is a good choice for scalable graphics that need to be resized without loss of quality.

  • Kind of graphical content: graphic, scalable to any size without loss of quality
  • Compression: varies (can be lossless or lossy)
  • 'Motion picture' support: no

MPG/MP4 (MPEG-1/MPEG-4):

MPG/MP4 is a good choice for motion pictures.

  • Kind of graphical content: motion picture
  • Compression: lossy
  • 'Motion picture' support: yes
